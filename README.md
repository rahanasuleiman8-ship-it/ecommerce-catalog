A modern e-commerce product catalog built with React, TypeScript, and Redux Toolkit.

## 🌐 Live Demo
**[https://ecommerce-catalog-rs.vercel.app](https://ecommerce-catalog-rs.vercel.app)**

## 🌟 Features
- **Product Catalog** - Browse products with images, prices, and descriptions
- **Category Filtering** - Filter products by category
- **Search Functionality** - Search products by name
- **Shopping Cart** - Slide-out drawer with add/remove/update quantities
- **State Persistence** - Cart saved to localStorage
- **Responsive Design** - Works on desktop and mobile

## 🛠️ Tech Stack
- **React 18** - UI library
- **TypeScript** - Type safety
- **Redux Toolkit** - State management (cart + products slices)
- **React Router** - Navigation
- **Tailwind CSS** - Styling
- **Fake Store API** - Product data
- **Vite** - Build tool
